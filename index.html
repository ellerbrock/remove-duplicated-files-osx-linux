Remove duplicated files on OS X and Linux

Tutorial how to find and delete duplicated files with fdupes on a UNIX based system.

OS X Installation:

brew update && brew install fdupes

Linux (apt based)

apt-get update && apt-get install fdupes

Usage

fdupes -r -d -N /folder/with/duplicated/files/ attention: deletes recursive directly!

-r recursive
-d delete
-N don't ask, delete directly
